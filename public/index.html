System Information API

This is a small API project for FCC where you can make a request and have the API send back to you some system information in JSON format for you to use should you want to.

Example Request

https://sysinfo-fcc.herokuapp.com/api

Example Response

{"ipaddress":"123.321.45.332","language":"en-US","software":"Windows NT 3.1; WOW64"}
